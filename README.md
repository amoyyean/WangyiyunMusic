# Wangyiyun
网易云音乐爬虫

**项目文件说明**：
本项目所有有关网易云的爬取采用的都是Scrapy框架。
## 一、WangyiyunUser文件
这个是网易云用户爬取项目文件
主文件为：WangyiyunSUer->WangyiyunUser->spiders->user.py
## 二、MusicMenu文件
这个是网易云音乐歌单项目文件
主文件为：MusicMenu->MusicMenu->Spiders->menu.py
## 三、wangyiyun文件
这个是网易云音乐歌曲《我想我不会爱你》评论爬取项目
主文件为：wangyiyun->wnagyiyun->spiders->musiccomments
